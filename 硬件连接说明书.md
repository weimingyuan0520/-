## 二、详细引脚分配表

### 2.1 光敏传感器 (模拟量 — 核心传感器)

| 功能 | ESP32 引脚 | 说明 |
|------|-----------|------|
| 光敏电阻模拟输出 | **GPIO5** (ADC1_CH4) | 接光敏电阻分压电路输出端，0~3.6V → 0~3600mV |
| VCC | 3V3 | 光敏模块供电 |
| GND | GND | 共地 |

> **电路建议**: 光敏电阻 (LDR) 与 10kΩ 固定电阻串联分压，中点接 GPIO5。  
> 光照越强，LDR 电阻越小，ADC 读数越高。

### 2.2 蜂鸣器

| 功能 | ESP32 引脚 | 说明 |
|------|-----------|------|
| 控制信号 | **GPIO6** | 高电平响、低电平停 |
| 有源蜂鸣器正极 | GPIO6 | 通过 NPN 三极管 (如 S8050) 驱动 |
| 有源蜂鸣器负极 | GND | |

> **驱动电路**:
> ```
> GPIO6 ──[1kΩ]──┬── 三极管基极 (B)
>                  │
>     3V3 ──[蜂鸣器]── 三极管集电极 (C)
>                  │
>                GND ── 三极管发射极 (E)
> ```

### 2.4 按键 (4 路，内部上拉)

| 按键 | ESP32 引脚 | 功能 | 默认状态 |
|------|-----------|------|---------|
| **K1** | **GPIO15** | 短按: LCD 页面切换；长按 2s: 进入/退出阈值配置模式 | 高电平 (释放) |
| **K2** | **GPIO17** | 配置模式下: 阈值减小 (短按-50, 长按-200) | 高电平 (释放) |
| **K3** | **GPIO8** | 配置模式下: 阈值增大 (短按+50, 长按+200) | 高电平 (释放) |
| **K4** | **GPIO40** | 预留 (当前未使用) | 高电平 (释放) |

> **接线**: 按键一端接 GPIO，另一端接 GND。代码已启用 GPIO 内部上拉，无需外接上拉电阻。

### 2.5 TFT LCD 液晶屏 (ST7789, 240×320, SPI 接口)

| LCD 引脚 | ESP32 引脚 | 信号说明 |
|----------|-----------|---------|
| GND | GND | 电源地 |
| VCC | 3V3 | 电源正 (3.3V) |
| **SCL** (SCK) | **GPIO21** | SPI 时钟线 |
| **SDI** (MOSI) | **GPIO47** | SPI 主出从入 (数据) |
| **SDO** (MISO) | **GPIO48** | SPI 主入从出 (可选) |
| **CS** | **GPIO42** | SPI 片选 |
| **RS** (DC) | **GPIO41** | 数据/命令选择 |
| **RST** | **EN** (系统复位) | LCD 复位 (与芯片复位共用) |
| **BL** | **GPIO14** | 背光控制 (高电平点亮) |
| TP_CS | **GPIO46** | 触摸屏片选 (预留) |
| TP_PEN | **GPIO2** | 触摸屏中断 (预留) |

> **注意**: 使用 SPI2_HOST (SPI2)，通信速率 20MHz。屏幕为竖屏模式 240×320。

### 2.6 串口 UART0 (连接 PC 调试)

| 功能 | ESP32 引脚 | 说明 |
|------|-----------|------|
| **TX** | **GPIO36** | 接 USB 转串口模块的 **RX** |
| **RX** | **GPIO38** | 接 USB 转串口模块的 **TX** |
| GND | GND | 共地 (必须!) |

> **参数**: 波特率 115200，8 数据位，1 停止位，无校验，无流控。  
> **推荐工具**: 工程根目录下 `串口调试助手\sscom33.exe`

### 2.6 LED 状态指示灯 (开发板板载)

| 功能 | ESP32 引脚 | 说明 |
|------|-----------|------|
| 红色 LED | **GPIO2** | 普中 ESP32-S3 AI 机器视觉开发板板载红色 LED，高电平点亮 |

> **状态**: 系统上电后常亮，表示设备正常运行。  
> **注意**: 此 LED 已集成在开发板 PCB 上，无需外接。GPIO2 同时为 ESP32 芯片的 strapping 引脚，上电时需保持低电平，启动后可正常使用。

### 2.7 继电器 (预留)

| 功能 | ESP32 引脚 | 说明 |
|------|-----------|------|
| 控制信号 | **GPIO10** | 高电平吸合，低电平断开 |

### 2.8 可调电阻 / 电位器 (调试用)

| 功能 | ESP32 引脚 | 说明 |
|------|-----------|------|
| 模拟输入 | **GPIO4** (ADC1_CH3) | 接电位器中点，用于调试和对比 |

### 2.9 PWM / 舵机接口 (预留)

| 功能 | ESP32 引脚 | 说明 |
|------|-----------|------|
| PWM 输出 | **GPIO4** | 50Hz PWM，舵机控制 (与 ADC_CH3 复用) |

## 四、电源要求

| 供电方式 | 电压 | 最大电流 | 说明 |
|----------|------|---------|------|
| USB 供电 | 5V | ≥500mA | 通过 ESP32-S3 开发板 USB 口供电，板载 LDO 转为 3.3V |
| 外部供电 | 5V | ≥500mA | 接开发板 5V / VIN 引脚 |

> ⚠️ **注意事项**:
> 1. 所有外设统一使用 3.3V 电平，**切勿直接接入 5V 信号**！
> 2. 蜂鸣器、继电器等大电流器件需通过三极管/驱动芯片隔离，不可直接用 GPIO 驱动。
> 3. USB 转串口模块的 VCC 可不接（由开发板 USB 供电即可），但 **GND 必须共地**。

## 五、接线步骤建议

### 第一步：最小系统验证
1. ESP32-S3 开发板连接 USB 供电
2. 烧录程序，观察板载红色 LED (GPIO2) 是否点亮
3. 打开串口助手 (115200bps)，观察启动日志

### 第二步：LCD 显示屏
1. 按 2.4 节表格依次连接 LCD 的 9 个引脚
2. 烧录后应看到 "Light Monitor" 标题画面

### 第三步：传感器 + 蜂鸣器
1. 光敏传感器模块接 3V3、GND、GPIO5 (仅需一个模拟光敏模块)
2. 蜂鸣器通过三极管接 GPIO6
3. 用手遮挡光敏电阻，验证蜂鸣器响应变化

### 第四步：按键
1. K1 (GPIO15)、K2 (GPIO17)、K3 (GPIO8) 各接一个按键到 GND
2. 按 K1 验证 LCD 页面切换
3. 长按 K1 进入阈值配置，K2/K3 调整阈值

### 第五步：串口通信
1. USB 转串口模块 RX → GPIO36 (ESP32 TX)
2. USB 转串口模块 TX → GPIO38 (ESP32 RX)
3. USB 转串口模块 GND → ESP32 GND
4. PC 端打开 `串口调试助手\sscom33.exe`，波特率 115200
5. 应收到 JSON 格式数据

### 第六步：WiFi 网络
1. 确保路由器 SSID 为 `666`，密码为 `15731491213`（或修改 `wifi-sta.h` 中的配置）
2. 烧录后 LCD 网络页面应显示 IP 地址
3. PC 端打开 `网络调试助手\NetAssist.exe`，TCP Client 模式连接该 IP:8080

## 六、快速故障排查

| 现象 | 可能原因 | 检查方法 |
|------|---------|---------|
| LED 不亮 | 供电问题 / 烧录失败 | 万用表测 3V3，检查串口日志 (板载 LED 为 GPIO2) |
| LCD 白屏 | SPI 接线错误 | 检查 SCL(GPIO21)、SDI(GPIO47)、CS(GPIO42)、RS(GPIO41) |
| LCD 黑屏 | 背光未开 | 检查 BL(GPIO14) 是否为高电平 |
| 光敏值一直为 0 | ADC 引脚错误 / 分压电路问题 | 测 GPIO5 电压，正常应在 0~3.3V 之间变化 |
| 按键无反应 | 未共地 / GPIO 接触不良 | 测 GPIO 电压，按下时应为 0V |
| 蜂鸣器不响 | 驱动电路问题 | 直接用 3V3 触碰蜂鸣器正极验证 |
| 串口无数据 | TX/RX 接反 / 波特率不对 | 交叉对调试一试，确认 115200 |
| WiFi 连不上 | SSID/密码不正确 | 检查 `wifi-sta.h`，修改 `DEFAULT_SSID` / `DEFAULT_PWD` |
